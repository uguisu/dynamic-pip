# dynamic-Pip功能介绍

`dynamic-Pip`能够帮助用户在运行时，轻松的对python包资源进行动态管理。

## 优势
- 在运行时对`package`进行动态添加/删除等管理
- 支持自动选择`PyPI`镜像站点，加快`package`安装速度
- 可生成`package`之间的依赖关系图，方便后续管理
- 仅依赖一个开源的网络速度测试库，对项目的侵入性非常小。避免造成版本冲突。
